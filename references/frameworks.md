# Strategy Frameworks Quick Reference

See `agents/strategist.md` for full framework instructions.

## Framework Selection Matrix

| Engagement Type | Must-Have Frameworks | Optional |
|---|---|---|
| Market Entry | Five Forces, PESTLE, Industry Life Cycle | CAGE (if international), VRIO |
| Competitive Strategy | VRIO, Productivity Frontier, Generic Strategies | Value Chain, Network Effects |
| M&A / Due Diligence | VRIO, Value Chain, Buy-Borrow-Build | Five Forces, Vertical Integration |
| International Expansion | CAGE, Cost-Responsiveness, PESTLE | Five Forces per target market |
| Digital / Platform Strategy | Network Effects, Business Model Archetypes, Crossing the Chasm | Value Creation & Capture |
| Full Strategy Review | AFI (full), Five Forces, VRIO, PESTLE | All others as relevant |
| Operational Turnaround | Value Chain, Productivity Frontier, Cost-Responsiveness | VRIO |
| New Venture / GTM | Crossing the Chasm, Business Model Archetypes, Industry Life Cycle | Five Forces |

## Frameworks at a Glance

| Framework | Core Question | Output |
|---|---|---|
| Porter's Five Forces | Is this industry attractive? | Industry attractiveness rating + implications |
| VRIO | Do we have sustainable competitive advantage? | Capability matrix with competitive implications |
| Productivity Frontier | Operational effectiveness vs. strategic positioning? | Position map + diagnosis |
| Generic Strategies | Where are we positioned? | Quadrant + "stuck in middle" risk assessment |
| AFI | Full strategic picture? | Analysis → Formulation → Implementation roadmap |
| PESTLE | What's the macro environment? | Environmental risk matrix |
| CAGE | How distant are international markets? | Distance matrix + market priority ranking |
| Value Chain | Where do we create/capture value? | Activity map + investment/outsource recommendations |
| Value Creation & Capture | Who captures the value and why? | Wedge analysis + pricing power assessment |
| Business Model Archetypes | What business model are we? | Archetype identification + transition pathway |
| Network Effects | Do we have network effects? | Network effects audit + critical mass analysis |
| Industry Life Cycle | Where is the industry? | Life cycle stage + stage-appropriate strategy |
| Crossing the Chasm | Can we cross to mainstream? | Chasm diagnosis + beachhead strategy |
| Buy-Borrow-Build | How do we close capability gaps? | Decision matrix per capability |
| Vertical Integration | Should we integrate? | Integration cost-benefit analysis |
| Cost-Responsiveness | Global or local strategy? | Quadrant position + organizational implications |
